| 2026-05-29 | analyser.minDecibels = −80 (narrow dB window to kill the silence floor) | Mapped a quiet iPhone mic's whole spectrum to byte 0 → dead (amplitude stuck at 0). Reverted to ~−100; gate via the windowed-min floor + the Floor slider |
| 2026-05-29 | EMA noise floor (down-fast 0.3 / up-slow 0.001) + prime-to-first-reading | Creeps up to the music level over a session AND primes high if the mic is enabled during sound → signal = max(0, energy−floor) → 0 → amplitude dies. Replaced by a windowed running-MINIMUM over ~8s (snaps to quiet, can't creep) |
| 2026-05-29 | Per-band DIRECTIONAL bursts (bass→radial centre, mid→L/R, treble→T/B velocity impulses) | Read as the mandala splitting into 4 axis-aligned quadrants — inorganic. Reverted to a single radial burst, then removed bursts entirely |
| 2026-05-29 | Radial velocity KICK (outward impulse on each beat) + the home spring | Two bugs in one: the kick pushes outward, the home pulls to the mandala origin. Removing home (for memoryless cymatics) left the kick unopposed → all sand piled at the disc edge into a ring ("eclipse"). Removed the kick; cymatics has no outward driver |
| 2026-05-29 | Home spring (vel += (origin−pos)·home) active during mic vibration | Forces grains back toward their start (mandala) position — not how cymatics works (memoryless: new = current + field). Set home=0 in mic mode; ❉ reform is the only deliberate origin pull |
| 2026-05-29 | ∇(f²) = −2f∇f field force + jitter ∝ \|f\| only | Force vanishes at nodes (fuzzy settling); jitter zero at nodes (perfect/deterministic lines). Replaced by crisp −∇\|f\| + a constant thermal jitter floor (stochastic, grainy lines) |

## Lessons
- Cymatic sand is a Langevin process: drift to nodes (constant-magnitude −∇|f|) + diffusion that never vanishes at the attractor; NO outward force, NO home/origin pull. Adding either an outward kick or a home spring alone drives particles to the boundary or freezes them. — from the eclipse + perfect-line dead ends on 2026-05-29
- "React to live music" is a CONTINUOUS envelope problem, not onset/beat detection — the latter misses sustained instruments and dense-mix kicks. — from the onset-detection dead end on 2026-05-29
- Adaptive silence gates that lerp toward the signal (EMA) creep up and silently kill reactivity over minutes; a windowed running-minimum (or a fixed user floor) is robust. — from the EMA-floor dead end on 2026-05-29
- iOS Web Audio: minDecibels can zero a quiet phone mic entirely; getByteFrequencyData's default −100/−30 window reads "silence" as ~0.28; AGC can't be disabled from the web (autoGainControl:false under-honored) — gate adaptively, don't over-clip. — from the −80dB dead end on 2026-05-29
